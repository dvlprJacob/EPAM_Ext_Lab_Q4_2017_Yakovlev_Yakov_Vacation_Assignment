Задание на каникулы : создать библиотеку структур данных, поэтапно добавлять новые структуры посредством
использования веток, тестировать его и выполнять слияние с основной веткой
Уточнение, в дальнейшем организация дерева веток будет следующая : master - ветка релиза
[master] <- develop - ветка разработки библиотеки, [master] <- [develop] <- queue,.... - ветки разработок соответствующих структур данных
